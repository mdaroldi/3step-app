# 3step-app - Simple React app to retrieve names from database.

## Technologies used:
### Web App:
* React.js
* React Router
* Bootstrap
* Stripe
* Seed
* JavaScript
* HTML
* CSS
* Node.js

### Backend - AWS:
* Lambda
* API Gateway
* DynamoDB
* Cognito
* CloudFront

## To run locally:
 * Clone repo
 * Run 'npm install'
 * Run 'npm start'
 * The app will automatically open in your default web browser, at localhost:3000
 * It is necessary to configure the file config.js before use
